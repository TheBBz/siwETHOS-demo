# React Overview

Beautiful, pre-built React components for Sign in with Ethos authentication.

## What's Included

The `@thebbz/siwe-ethos-react` package provides:

### Components
- **EthosAuthModal** - Complete authentication modal with wallet and social login
- **SignInButton** - Customizable sign-in button
- **Other UI components** - Supporting components for custom implementations

### Hooks
- **useEthosAuth** - Manage authentication state
- **Other hooks** - Additional hooks for common patterns

### Styling
- Pre-built CSS with dark/light mode
- Customizable with CSS variables
- Full theme control

## Quick Start

```tsx
import { useState } from 'react';
import { EthosAuthModal } from '@thebbz/siwe-ethos-react';
import '@thebbz/siwe-ethos-react/styles.css';

function App() {
  const [isOpen, setIsOpen] = useState(false);
  const [user, setUser] = useState(null);

  return (
    <div>
      <button onClick={() => setIsOpen(true)}>
        {user ? `Welcome ${user.profile.displayName}` : 'Sign In'}
      </button>

      <EthosAuthModal
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        onSuccess={(result) => {
          setUser(result);
          setIsOpen(false);
        }}
      />
    </div>
  );
}
```

- MetaMask
- Rabby
- Phantom
- Zerion

- Brave Wallet
- Any EIP-1193 compatible wallet

### Social Authentication
- Farcaster (QR code)
- Discord (OAuth2)
- Twitter/X (OAuth2)
- Telegram (Login Widget)
- GitHub (OAuth2)

## TypeScript Support

Full TypeScript support with comprehensive type definitions:

```tsx
import type {
  EthosAuthResult,
  EthosAuthModalProps,
  EthosProvider,
} from '@thebbz/siwe-ethos-react';

const handleSuccess = (result: EthosAuthResult) => {
  console.log('Provider:', result.provider);
  console.log('Profile:', result.profile);
};
```
