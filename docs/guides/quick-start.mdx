# Quick Start Guide

Get up and running with Sign in with Ethos in minutes!

## Choose Your Path

### üöÄ React (Easiest)

Perfect for React applications. Get a beautiful authentication modal with minimal code.

#### 1. Install

```bash npm2yarn
npm install @thebbz/siwe-ethos-react
```

#### 2. Add to Your App

```tsx
import { useState } from 'react';
import { EthosAuthModal } from '@thebbz/siwe-ethos-react';
import '@thebbz/siwe-ethos-react/styles.css';

function App() {
  const [isOpen, setIsOpen] = useState(false);
  const [user, setUser] = useState(null);

  return (
    <div>
      <button onClick={() => setIsOpen(true)}>
        {user ? `Welcome ${user.profile.displayName}` : 'Sign In'}
      </button>

      <EthosAuthModal
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        onSuccess={(result) => {
          setUser(result);
          setIsOpen(false);
        }}
      />
    </div>
  );
}
```

#### 3. Done! üéâ

You now have:

- ‚úÖ Social logins (Discord, Twitter, Farcaster, etc.)
- ‚úÖ Beautiful UI with dark mode

### üì¶ SDK (Vanilla JS/TS)

For non-React apps or custom implementations.

#### 1. Install

```bash npm2yarn
npm install @thebbz/siwe-ethos
```

#### 2. Authenticate

```javascript
import { EthosWalletAuth } from '@thebbz/siwe-ethos';

const auth = EthosWalletAuth.init();

// With wagmi
import { useAccount, useSignMessage } from 'wagmi';

const { address } = useAccount();
const { signMessageAsync } = useSignMessage();

const result = await auth.signIn(address, signMessageAsync);

console.log('User:', result.user.name);
console.log('Score:', result.user.ethosScore);
console.log('Token:', result.accessToken);
```

#### 3. Store Token

```javascript
localStorage.setItem('ethos_token', result.accessToken);

// Later, retrieve user
const token = localStorage.getItem('ethos_token');
const user = await auth.getUser(token);
```
