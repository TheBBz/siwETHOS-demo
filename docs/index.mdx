# Sign in with Ethos Documentation

Welcome to the **Sign in with Ethos** documentation! This project provides authentication solutions that leverage the [Ethos Network](https://ethos.network) for decentralized reputation-based authentication.

## What is Sign in with Ethos?

Sign in with Ethos enables developers to authenticate users using their Ethereum wallets (SIWE - Sign-In with Ethereum) or social accounts (Discord, Twitter, Farcaster, Telegram, GitHub), while simultaneously accessing their Ethos credibility scores and reputation data.

## Installation

Choose the package that fits your needs:

```bash
# For vanilla JavaScript/TypeScript projects
npm install @thebbz/siwe-ethos

# For React applications
npm install @thebbz/siwe-ethos-react

# For custom provider implementations
npm install @thebbz/siwe-ethos-providers
```

## Quick Example

### React

```tsx
import { EthosAuthModal } from '@thebbz/siwe-ethos-react';
import '@thebbz/siwe-ethos-react/styles.css';

function App() {
  const [isOpen, setIsOpen] = useState(false);

  const handleSuccess = (result) => {
    console.log('Authenticated:', result.profile.displayName);
    console.log('Ethos Score:', result.profile.score);
  };

  return (
    <>
      <button onClick={() => setIsOpen(true)}>Sign In</button>
      <EthosAuthModal 
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        onSuccess={handleSuccess}
      />
    </>
  );
}
```

### SDK

```javascript
import { EthosWalletAuth } from '@thebbz/siwe-ethos';

const auth = EthosWalletAuth.init();

// Authenticate with wallet
const result = await auth.signIn(walletAddress, signMessageAsync);
console.log('User:', result.user.name);
console.log('Score:', result.user.ethosScore);
```
